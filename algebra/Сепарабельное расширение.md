**Сепара́бельное расширение** — [[Расширение поля|алгебраическое расширение]] [[Поле|поля]] $E \supset  K]$, состоящее из сепарабельных элементов, то есть таких элементов $\alpha$, минимальный [[Аннулятор|аннулятор]] $f(x)$ над $K$ для которых не имеет кратных корней. Производная $f'(x)$ должна быть в этой связи ненулевым многочленом. По определению все поля [[Характеристика|характеристики]] 0 сепарабельны, поэтому понятие сепарабельности нетривиально лишь для полей ненулевой характеристики $p$.
