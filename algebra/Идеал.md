Пусть A — произвольное кольцо. Подмножество I ⊂ A называется __идеалом__, если выполнено следующее: I
 1. : ∀x, y ∈ I x + y ∈ I; 
 2. : ∀x ∈ I −x ∈ I; 
 3. : ∀x ∈ I, a ∈ A верно, что ax ∈ I.
 Первые две аксиомы равносильны тому, что I — подгруппа в A по сложению (в частности, 0 ∈ I). На самом деле вторая аксиома следует из третьей: т.к. −1 ∈ A, то I оказывается замкнуто по умножению на −1: то есть если x ∈ I, то −x ∈ I. 
 
 Обратите внимание, что в третьей аксиоме требуется замкнутость I по умножению на все элементы из A (а не только из I) — это очень существенное ограничение. В частности, если $1\in I$ то I =  A: единицу можно умножить на любой элемент кольца, и результат будет снова принадлежать I. По той же причине идеал, содержащий любой обратимый элемент, совпадает со всем кольцом.
 
 Идеал вида (a) ⊂ A, т.е. порождённый одним элементом, называется __главным идеалом__
 
 Кольцо называется __целостным __ (или областью целостности), если в нём произведение любых двух ненулевых элементов отлично от нуля (т.е. из того, что ab = 0, следует, что либо a = 0, либо b = 0). 
 
 Целостное кольцо A называется __кольцом главных идеалов__, если все идеалы в нём являются главными.
 
#### Теорема
Z и K\[x\] — кольца главных идеалов. 
##### Доказательство. 
Пусть I — ненулевой идеал в Z. Выберем в нём наименьший положительный элемент и обозначим его через d. Докажем, что I = (d). Действительно, пусть x ∈ I. Разделим x на d с остатком: x = dq + r. Значит, r = x − dq. Оба слагаемых в правой части принадлежат I, значит, r ∈ I. Но 0 ≤ r < d. Поэтому r = 0, и x делится на d нацело. Значит, x ∈ (d). Случай K\[x\] разбирается аналогично, только в качестве d надо взять элемент наименьшей степени.
